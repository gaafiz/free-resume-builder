Chose which resume sections will be shown in the left or right columns. Valid Sections are:
  • photo (display picture of the candidate)
  • labeling (display name and professional role of the candidate)
  • summary (display a brief summary about the candidate)
  • personalInfo (display personal info and contacts of the candidate)
  • languages
  • skills
  • education
  • experience
Left Column Sections Enter sections separated by semicolon ';'
Right Column Sections Enter sections separated by semicolon ';'
Here you can change (or remove) the title for the following sections:
"{{section.key}}" section Enter blank for no title

By Saving the Layout Configuration in a file, you may apply it back any time.
